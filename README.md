# Math Color

## Overview
Math Color is a plugin for Obsidian that enhances your mathematical notation by allowing you to color selected math expressions.

## Features
- Color any math expression with your choice of color.
- Easy integration into your existing Obsidian setup.

## Installation
1. Download the plugin files.
2. Move them into your Obsidian plugins directory.
3. Enable the plugin through the Obsidian settings.

## Usage
Simply select a math expression and use the specified command to apply your chosen color!
